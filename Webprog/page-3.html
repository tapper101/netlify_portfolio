Uppgift 4 - Validering med HTML och JavaScript

WEBPROG
ABSTRAKT:

Skapa ett formulär i HTML och sedan validera med HTML och JavaScript

LÖSNING:

Först skapade jag ett simpelt formulär i HTML som jag sedan stylade med CSS.

Själva valideringen skedde främst via regular expressions i både HTML och JavaScript, samt med HTML-attributet 'required' som kollar om fältet överhuvudtaget har fyllts i. Den stora utmaningen var att validera BIC-numret, vilket jag löste genom att först kontrollera (if)all det givna värdet följer det mönster som BIC-nummer ska ha. Om det gjorde det så loopade jag genom värdet 10 gånger för att få ihop summan som sedan ska divideras med 11 - vilket i sin tur ska lämna en restsumma som motsvarar kontrollsiffran.

Emailadressens domän jämfördes med det valda universitet (mer specifikt dess värde) i JavaScript.
